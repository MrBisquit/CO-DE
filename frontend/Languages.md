# Supported languages

| Language   | File Extensions (Aliases) | Supported |
| ---------- | ------------------------- | --------- |
| TypeScript | *.ts, *.tsx               | Yes       |
| JavaScript | *.js                      | Yes       |
| C#         | *.cs                      | No        |
| C          | *.c                       | No        |
| C++        | *.cpp                     | No        |
| CSS        | *.css                     | Yes       |
| JSON       | *.json                    | Yes       |
| SVG        | *.svg                     | Yes       |
| Haskell    | *.hs                      | No        |
| Python     | *.py, *python3            | No        |
